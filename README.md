# Jahresplaner

That's all, a year planner:
- interactive
- local only
- automatic counting
- automatic light/dark following system theme
- with my personal defaults

## Run

Serve `index.html` via a small Go HTTP server on port 2026:

```bash
go run main.go
```

Then open http://localhost:2026
